# Graduation_Project
제목: 화장품 쇼핑몰 고객 데이터 활용 이탈 요인 분석 및 데이터 기반 마케팅 기획 서비스

배경: 주제를 선택한 이유 (달바 글로벌 회사에서 브랜드 마케팅 물류 업무를 진행하며 데이터로 문제를 분석하는 과정에 흥미를 느낌, 최근 트렌드에 민감하고 재구매 주기가 명확한 분야인 코스메틱 분야에서 이탈 징후를 포착하고 맞춤형 추천을 제공해서 고객 유지율을 높이는 방법을 제시하고자 함

활용 언어 : SQL / Python(Pandas, Scikit-learn), Streamlit

위 프로젝트로 인한 목표: 이 서비스를 통해 쇼핑몰 매출이 어떤 방향성으로 오를 수 있는지에 관한 건
